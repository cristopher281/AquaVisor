# Carpeta de Imágenes

Esta carpeta contiene las imágenes utilizadas en la documentación y recursos del repositorio AquaVisor.

## Estructura Sugerida

- **capturas/**: Screenshots de la aplicación web
- **diagramas/**: Diagramas de arquitectura y flujo
- **hardware/**: Fotos del ESP32 y configuración de hardware
- **diseño/**: Mockups y diseños de la interfaz

## Uso

Las imágenes en esta carpeta pueden ser referenciadas en la documentación usando rutas relativas:

```markdown
![Descripción](../images/nombre-imagen.png)
```

## Formato Recomendado

- Utiliza formatos **PNG** para capturas de pantalla y diagramas
- Utiliza formatos **JPG** para fotos
- Mantén nombres descriptivos en minúsculas con guiones (ej: `dashboard-principal.png`)
